The Solution

STRUCTURED
TOUCH.

Instead of adding more sound, we opened an entirely new sensory channel. The forehead has dense nerve endings, a large cortical representation, and — crucially — it leaves both hands and ears completely free.

A 3×8 matrix of 24 vibration actuators encodes real-time 3D spatial data into a tactile language. 160 unique patterns. Object identity, distance, direction — all delivered in under 330ms.

01 — SENSE
DEPTH
Intel RealSense D435i. Stereo IR. IMU stabilization. True metric depth at 30fps.
02 — DETECT
VISION
YOLOv8 real-time object detection. 80+ classes identified with bounding boxes.
03 — ENCODE
LANGUAGE
160 unique haptic patterns. Depth-modified by proximity zone. Simultaneous multi-object.
04 — FEEL
TOUCH
3×8 LRA actuator grid. DRV2605L haptic drivers. I²C multiplexed. <10ms actuation.

Technical Pipeline

CAMERA
TO FOREHEAD

Five stages. Under 330 milliseconds. Real time.

01
CAMERA
Intel RealSense D435i. 640×480 RGB+Depth. IMU stabilizes head-motion in real time.
02
DETECT
YOLOv8 identifies 80+ object classes. Alternates Object/Depth mode every 1 second.
03
ENCODE
Object → pattern ID. Depth zone → modifier (+0 to +3). 160 unique patterns total.
04
TRANSMIT
JSON array via USB CDC serial at 115200 baud to 3× QT Py SAMD21 microcontrollers.
05
ACTUATE
DRV2605L haptic ICs drive LRA motors via I²C mux. <10ms actuation. 24 nodes.

Hardware Evolution

17 VERSIONS.
ONE GOAL.

V1–V5
STRUCTURAL FEASIBILITY
Initial prototypes used simple 3D-printed brackets and elastic headbands. Functional but suffered from haptic bleeding — vibrations from one motor propagated across the entire rigid plastic frame, destroying spatial resolution.
MaterialPLA + Elastic
IssueHaptic bleed
StatusDeprecated
V6–V9
CURVATURE BREAKTHROUGH
Frame modeled around average human forehead curvature. Rubber-seated architecture introduced — mechanically isolating each motor so vibrations stay localized. A pulse on the left is clearly distinct from center. Spatial resolution restored.
MaterialPLA + Rubber
WinVibration isolation
StatusKey milestone
V10–V16
HYBRID MATERIAL SYSTEM
TPU matrix for the haptic actuator housing — flexible, vibration-isolating, skin-safe. PLA main housing for rigidity. Integrated cable channels prevent snagging. Reinforced camera mount maintains precise depth-sensing alignment.
MaterialTPU + PLA
WinCable management
StatusNear-final
V17
FINAL PRODUCTION DESIGN
Optimized camera mount alignment. Improved dowel mounting system. Bridge architecture with silicone-attached bridges allows tool-free matrix removal without disturbing wiring. Velcro-secured bridges. 246g total weight. Wearable for extended periods.
